# Statistical Analysis of Diabetes Risk Factors

## Overview
This project investigates the relationship between major health indicators and diabetes risk using statistical analysis and exploratory data analysis techniques in R.

The analysis focuses on variables such as:
- HbA1c levels
- Blood glucose levels
- BMI
- Age

The goal is to identify risk-associated patterns and examine predictive relationships with diabetes prevalence.

---

## Tools Used
- R
- RStudio
- ggplot2
- dplyr
- tidyr
- readr

---

## Methods Used
- Exploratory Data Analysis (EDA)
- Summary Statistics
- Distribution Analysis
- Spearman Correlation Analysis
- Wilcoxon Rank-Sum Tests
- Logistic Regression
- Data Visualization
